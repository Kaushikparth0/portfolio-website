## About The Project

This portfolio website serves as a central hub for my professional journey, featuring my technical skills, education, experience, and real-world projects.

It highlights:

* My technical skillset
* Full-stack development projects
* Project descriptions with deployed links
* Education and experience
* Contact information for collaboration opportunities

The website is designed with a clean UI, smooth navigation, and responsive layout to provide a seamless experience across devices.

## Installation & Setup

Clone the repository:

```bash
git clone <repository-url>
```

Navigate to project folder:

```bash
cd react-tailwind-personal-portfolio
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```
